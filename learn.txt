<a href="http://php.net/manual/en/function.addslashes.php" target="_blank">addslashes(strSUBJECT)</a>	Escapses the special characters (like '"' & '$') with '\' in a string strSUBJECT and returnes the resulting escaped string. Be careful, this escapes both the single and double quotes with slash.
<a href="http://php.net/manual/en/function.strcasecmp.php" target="_blank">strcasecmp(strFIRST, strSECOND)</a>	Compares two strings (case insensitive & binary safe). Returns < 0 if strFIRST is less than strSECOND; > 0 if strFIRST is greater than strSECOND, and 0 if they are equal.
<a href="http://php.net/manual/en/function.explode.php" target="_blank">explode(strSEPERATOR, strDATA)</a>	Cuts down a bigger string strDATA into pieces by string strSEPERATOR and returns all the pieces in an array.
<a href="http://php.net/manual/en/function.get_defined_functions.php" target="_blank">get_defined_functions()</a>	Returns an array of both PHP's internal and user defined functions currently available in the installed system.
<a href="http://php.net/manual/en/function.implode.php" target="_blank">implode(strSEPERATOR, arrDATA)</a>	Joins all the items of an array arrDATA with string strSEPERATOR and returns as a string.
<a href="http://php.net/manual/en/function.require.php" target="_blank">require(strFILE)</a>	Executes another PHP script strFILE in the middle of the current script where called as if all of them were a single script.
